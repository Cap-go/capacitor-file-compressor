# Example App for `@capgo/capacitor-file-compressor`

This Vite project links directly to the local plugin source so you can exercise the native APIs while developing.

## Actions in this playground

- **Get plugin version** – Calls getPluginVersion() to retrieve the plugin version.
- **Select image** – Allows selecting an image from the device.
- **Compress image** – Compresses the selected image with adjustable quality and dimensions.
- **Preview result** – Displays the compressed image and shows file size reduction.

## Getting started

```bash
npm install
npm start
```

Add native shells with `npx cap add ios` or `npx cap add android` from this folder to try behaviour on device or simulator.
